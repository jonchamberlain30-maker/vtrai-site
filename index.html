Common Questions

These are the practical questions a new visitor usually needs answered before they decide whether Vectora is worth revisiting.

What does Vectora actually do?

It checks token trust across major retail chains, shows the main proof links, and gives users a cleaner decision point before they spend more time researching.

Does it replace DexScreener?

No. DexScreener is still where most discovery and chart checking happens. Vectora is the next step: route verification, trust checks, watchlists, and monitoring.

What does the watchlist do?

The watchlist saves tokens on this browser, refreshes trust reports on demand, and keeps a local change history so the user can see whether a token is drifting.

What is scheduled monitoring?

Scheduled monitoring moves selected tokens into shared background checks so new trust snapshots can appear on the updates desk even when the user is not manually refreshing them.
